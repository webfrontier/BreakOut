インターン課題案
　・初期状態はブロック1列のみ

ーーー　基本部分　−−−

　１．起動確認（ビルド、デバッグなどの操作を覚える）
　　　◎全員が起動できたら、まずフレーム処理（1/60で処理がされるなどの話をする）
x　２．バーの表示
x　３．バーの移動速度の調整
x　４．ボタンの表示（まずは、”Start”のみ）
x　５．ボールを表示（ただし、バーとの当たり判定なし）
x　６．ボールが遅いので速度調整
x　７．バーとボールの当たり判定追加（画面端での折り返しなし）
x　８．画面端でのボールの跳ね返り判定修正
x　９．ボールとブロックの当たり判定追加
x　１０．ブロックを全て消したら、クリア画面を出す
x　１１．ボールが下に落ちるとゲームオーバー画面
x　１２．ブロックの列を追加する
x　１３．BLEデバイスの接続

ーーー　機能追加部分　−−−

x　１４．ブロックに耐久度を追加
x　１５．壊れないブロックの追加
　１６．特定のブロックに当たると、ボールが増える
　１７．スタートから一定時間経つとボールのスピードが上がる
　１８．ブロックに当たると速度が加速する
　１９．バーが伸びる（短くなる）
　２０．特定のブロックのみ跳ね返りが違うようにする
　２１．ブロックを壊すとアイテムが落ちてくるようにする
x　２２．ブロックの配置を考える
　２３．効果音の追加
　２４．スコア表示

ーーー　できる学生用部分　−−−

　２５．ブロックを壊すとスコア加算、ブロックに当たらず戻ってくるとスコア減算
　２６．ボールの跳ね返り挙動を考える（放物線など）
　２７．画面をタップするとバーが弾を撃つようにする
　２８．ブロックから弾が降ってくるようにする（当たると死ぬ）
　２９．ブロックが動く
　３０．ボールに追従しておじゃまブロックが動く
